## Installation



1. Download this repo

2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your BOT Token in `config.json`
   ```js
    BOT_TOKEN = 'ENTER YOUR TOKEN'
   ```


## Usage
Execute the command ``!setup`` to post the embed.
